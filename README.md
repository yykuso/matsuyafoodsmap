# matsuyafoodsmap

松屋フーズグループの店舗情報を地図で可視化し、ブランド別や店舗限定メニュー実施店舗で絞り込めるWebアプリです。

## 主な機能

- 全国の店舗を地図上にプロットして表示
- ブランドごとに色分けした店舗表示
- ブランドフィルターによる絞り込み
- 店舗限定メニューCSVによる絞り込み
- 実施中/過去メニューを選択して対象店舗のみ表示
- 店舗クリックでポップアップ表示
- Googleマップ/Appleマップ検索リンクをポップアップに表示
- 同一座標の店舗が重なりにくいように座標オフセット
- 地図の中心座標とズームをローカル保存して次回復元
- ブランド/店舗限定フィルターの選択状態をローカル保存して次回復元
- 現在地ボタン、ズーム操作、スケール表示
- スマートフォン向けレイアウト対応

## 使い方

1. ローカルサーバーでプロジェクトを配信します。
2. ブラウザでトップページを開きます。
3. 右上の「ブランド」「店舗限定」セレクトで表示対象を絞り込みます。
4. 店舗をクリックして外部地図アプリへの検索リンクを利用します。

ローカル起動例:

```bash
python -m http.server 8000
```

## データ構成

- `data/matsuyafoods.csv`
	- 全店舗の基本データ
	- 主な列: `code`, `name`, `brand`, `latitude`, `longitude`
- `data/*.csv`（キャンペーン/店舗限定メニュー）
	- 1列目に店舗コードを持つCSV
	- 選択したCSVの店舗コードに一致する店舗のみを表示

## 技術スタック

- HTML/CSS/Vanilla JavaScript
- MapLibre GL JS（地図表示）
- Papa Parse（CSV読み込み）
